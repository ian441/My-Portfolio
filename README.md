# Ian.ma Portfolio

This is a personal portfolio website showcasing my skills, experience, projects, and contact information. The site is built with HTML, CSS, and JavaScript.

## Features

- Responsive design for all devices
- About section with skills, experience, and education tabs
- Services section
- Projects section with featured project and tech stack
- Like button with persistent count (localStorage)
- Contact form and social links

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/your-portfolio-repo.git
   ```
2. **Open the project folder:**
   ```
   cd your-portfolio-repo
   ```
3. **Open `index.html` in your browser.**

## Project Structure

```
index.html
style.css
script.js
images/
```

- `index.html` – Main website content
- `style.css` – All styles and responsive design
- `script.js` – Tab switching, project filtering, modal, like button logic
- `images/` – Profile and project images

## Customization

- Update your information in `index.html`
- Replace images in the `images/` folder
- Add more projects in the Projects section
